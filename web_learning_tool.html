📝 Lesson 1: Variables & Print Statements

Variables are like containers that store information. Think of them as labeled boxes where you can put things!

name = 'Python Learner' age = 25 favorite_color = 'blue' print('Hello, ' + name) # String concatenation print(f'You are {age} years old') # f-strings (modern way) print('Your favorite color is', favorite_color) # Multiple values

🎯 Lesson 3: Control Flow (if/else statements)

Control flow lets your program make decisions! It's like giving your computer a brain to choose what to do.

if temperature > 25: print("It's warm! Let's make an iced coffee!") elif temperature > 15: print("Nice weather! A regular coffee sounds good!") else: print("It's chilly! Let's make a hot coffee!")

⚙️ Lesson 4: Functions

Functions are like recipes - they package code into reusable blocks! They help you organize your code and avoid repetition.

def greet_person(name, time_of_day="day"): if time_of_day == "morning": return f"Good morning, {name}! ☀️" elif time_of_day == "evening": return f"Good evening, {name}! 🌙" else: return f"Hello, {name}! 👋"

📋 Lesson 6: Lists & Data Structures

Lists are like shopping lists - they hold multiple items! They're perfect for storing collections of data.

🛡️ Lesson 7: Error Handling

Sometimes things go wrong in programs - that's normal! Error handling helps your program deal with problems gracefully.

try: number = int(input("Enter a number: ")) print(f"Great! You entered: {number}") except ValueError: print("That's not a valid number! Try again.")
