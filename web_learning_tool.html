Variables & Print Statements

Learn how to store data and display output

Variables are containers that store information. They allow you to save data and use it throughout your program.

name = 'Python Learner' age = 25 favorite_color = 'blue' print('Hello, ' + name) # String concatenation print(f'You are {age} years old') # f-strings (modern way) print('Your favorite color is', favorite_color) # Multiple values

Get information from users and build interactive programs

The input() function allows you to ask questions and get answers from users. This makes your programs interactive.

Control Flow

Make decisions with if/else statements

Control flow allows your program to make decisions based on conditions. It's like giving your computer a brain to choose what to do.

if temperature > 25: print("It's warm! Let's make an iced coffee!") elif temperature > 15: print("Nice weather! A regular coffee sounds good!") else: print("It's chilly! Let's make a hot coffee!")

Functions

Create reusable code blocks

Functions package code into reusable blocks. They help you organize your code and avoid repetition.

def greet_person(name, time_of_day="day"): if time_of_day == "morning": return f"Good morning, {name}!" elif time_of_day == "evening": return f"Good evening, {name}!" else: return f"Hello, {name}!"

Error Handling

Handle errors gracefully

Error handling helps your program deal with problems gracefully. It prevents crashes and provides better user experience.

try: number = int(input("Enter a number: ")) print(f"Great! You entered: {number}") except ValueError: print("That's not a valid number! Try again.")
